Class WriteCommand
Command to write text, numeric expressions, or method return values onto the canvas at the current pen position.
Syntax
C#
    internal class WriteCommand : CanvasCommand
Remarks
Resolution priority for the parameter: 1. Method return values via GetReturnValue(string). 2. BOOSE program variables or expressions. 3. Literal text if no variable or method exists.
