Class SimpleParser
A lightweight parser that executes BOOSE-style commands directly, bypassing the main BOOSE parser.
Syntax
C#
    internal class SimpleParser
Remarks

Purpose: This class allows "direct execution" of scripts with full support for custom method calls, loops, and canvas operations without using the built-in BOOSE interpreter.

Features:

  • Supports int and real variable declarations.
  • Supports array creation, poke, and peek operations.
  • Executes for loops, while loops, and if statements directly.
  • Supports method definition and calling via MethodCommand.
  • Executes drawing commands: moveto, drawto, circle, rect, write, pen/pencolour, etc.

Usage Example:

C#
VariableStore store = new VariableStore();
SimpleParser parser = new SimpleParser(store);
parser.SetCanvas(myCanvasInstance);
parser.ExecuteProgram(@"
    int x = 10
    int y = 20
    circle x
");
