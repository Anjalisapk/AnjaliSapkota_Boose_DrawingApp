Class CircleCommand
Implements the circle drawing command for the BOOSE language. Draws an unfilled circle with the specified radius at the current pen position.
Syntax
C#
    internal class CircleCommand : CanvasCommand
Remarks
Example usage in a BOOSE script:
C#
moveto 100,100
circle 50
