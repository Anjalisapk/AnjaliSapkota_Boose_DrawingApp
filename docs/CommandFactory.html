Class MyCommandFactory
Factory class responsible for creating BOOSE command objects based on the command name parsed by the interpreter.
Syntax
C#
    internal class MyCommandFactory : CommandFactory
Remarks
This factory is registered inside the BOOSE StoredProgram and is used every time the interpreter encounters a command keyword.

Method MakeCommand

Parameters

The BOOSE command keyword (e.g., moveto, circle).
Return Value
An instance of a class implementing ICommand, which represents the executable BOOSE command.
Exceptions
