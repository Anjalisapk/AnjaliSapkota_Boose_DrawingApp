Class PenColourCommand
Represents a command to set the pen color in the BOOSE canvas. Inherits from CommandThreeParameters and requires three integer parameters: R, G, B.
Syntax
C#
    public class PenColourCommand : CommandThreeParameters
Remarks
This command sets the current drawing color for the canvas. RGB values must be integers in the range 0-255.

Example usage in BOOSE:

C#
pen 255 0 0    // sets color to red
pencolour 0 255 0 // sets color to green

Throws a CanvasException if any value is a double or out of range.
