Class AppCanvas
Represents the drawable canvas used by BOOSE commands. Handles pen movement, drawing operations, colours, and bitmap output.

Design Pattern: Singleton Pattern This class implements the Singleton design pattern to ensure only one canvas instance exists throughout the application lifecycle. This prevents conflicts when multiple commands try to draw on different canvas instances.

Design Pattern: Adapter Pattern This class acts as an adapter between the BOOSE framework's ICanvas interface and the .NET System.Drawing API, allowing BOOSE commands to work with Windows Forms graphics without knowing the underlying implementation details.

Syntax
C#
    public class AppCanvas : ICanvas
Remarks
The canvas uses a Bitmap and Graphics object to perform all drawing operations. The pen position (Xpos, Ypos) tracks the current drawing cursor location.

Method DrawTriangle
Draws a triangle at the current pen location.
Syntax
C#
    public void DrawTriangle(int width, int height)
Parameters
width
Base width of the triangle.
height
Height of the triangle.
Remarks

The triangle is drawn with the following vertices:

  • Top vertex: (Xpos + width/2, Ypos)
  • Bottom-left: (Xpos, Ypos + height)
  • Bottom-right: (Xpos + width, Ypos + height)

This creates an isosceles triangle with the base at the bottom.
