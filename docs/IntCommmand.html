Class IntCommand
command for integer variable declarations. Actual processing is done by SimpleParser.

Syntax:int variableName [= value]

Example:

C#
int radius = 50
int count

Syntax
C#
    internal class IntCommand : CanvasCommand
Remarks
This class exists to satisfy the Factory Pattern. The SimpleParser intercepts 'int' declarations before they reach the BOOSE parser, so this Execute method is typically not called.
