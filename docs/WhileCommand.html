Class WhileCommand
Represents a while loop command in BOOSE. Evaluates a condition repeatedly until it becomes false.
Syntax
C#
    internal class WhileCommand : CanvasCommand
Remarks
Example usage:
C#
while height > 50
    moveto x,y
    drawto x2,y2
endwhile
The actual looping logic is handled by the BOOSE engine; this command only evaluates the loop condition.
