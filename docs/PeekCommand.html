Class PeekCommand
Command for retrieving values from arrays (peek operation). Actual processing is done by SimpleParser and VariableStore.

Syntax:peek variableName = arrayName index

Example:

C#
array int scores 5
poke scores 0 = 100
peek myScore = scores 0
circle myScore

Syntax
C#
    internal class PeekCommand : CanvasCommand
Remarks
The 'peek' command retrieves a value from a specified index in an array and stores it in a variable. Arrays are zero-indexed.
