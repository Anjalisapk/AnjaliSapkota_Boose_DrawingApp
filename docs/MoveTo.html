Class MoveToCommand
Represents the BOOSE moveto command, which moves the pen to a specified (X,Y) coordinate on the drawing canvas without drawing a line.

Syntax: moveto x,y

This command:
  • Accepts literal numbers or BOOSE expressions.
  • Updates pen position without rendering graphics.
  • Relies on MoveTo(int, int) for movement.
  • Validates parameters for correct comma-separated formatting.

Method CheckParameters

Method Set
Configures the moveto command by parsing and evaluating the supplied coordinate expressions. Supports literal integers and full BOOSE arithmetic expressions.

Example inputs: moveto 100,200moveto width/2, height-10

Syntax
C#
    public override void Set(StoredProgram program, string Params)
Parameters
program
The StoredProgram instance executing this command.
Params
A comma-separated string containing X and Y expressions.
Exceptions
Exception Condition
ArgumentException Thrown when:
  • Parameters cannot be split into exactly two values.
  • An expression cannot be evaluated into a number.
  • The final X or Y values cannot be parsed as integers.
