Class PokeCommand
Command for storing values in arrays (poke operation). Actual processing is done by SimpleParser and VariableStore.

Syntax:poke arrayName index = value

Example:

C#
array int scores 5
poke scores 0 = 100
poke scores 1 = 95

Syntax
C#
    internal class PokeCommand : CanvasCommand
Remarks
The 'poke' command stores a value at a specified index in an array. Arrays are zero-indexed (first element is index 0).
