Class TriangleCommand
Represents the BOOSE triangle command. Draws an unfilled triangle at the current pen position with specified width and height.

Syntax:triangle width,height

Design Pattern: This class follows the Command Pattern, encapsulating a drawing operation as an object. It inherits from CommandTwoParameters which is part of the BOOSE framework's command hierarchy.

Example Usage:

C#
moveto 100,100
triangle 50,60
This will draw a triangle with base width 50 and height 60 at position (100,100).

The triangle is drawn with: - Top vertex at (Xpos + width/2, Ypos) - Bottom-left vertex at (Xpos, Ypos + height) - Bottom-right vertex at (Xpos + width, Ypos + height)

Syntax
C#
    public class TriangleCommand : CommandTwoParameters
Remarks
Related Design Patterns:
  • Command Pattern: Each drawing command is encapsulated as an object
  • Factory Pattern: Created by MyCommandFactory
  • Singleton Pattern: Uses AppCanvas singleton instance

Exceptions
Exception Condition
CanvasException Thrown when:
  • Double parameters are provided (only integers supported)
  • Width or height is less than 1
Remarks

Validation Rules:

  1. Only integer values are accepted (no doubles)
  2. Both width and height must be greater than 0

Drawing Process: The triangle is drawn by calling Tri(int, int) which connects three points to form a triangle shape.
