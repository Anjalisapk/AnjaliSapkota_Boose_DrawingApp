Class ElseCommand
Dummy command for the 'else' keyword in conditional statements.

Design Pattern Note: This is part of the Factory Pattern implementation. Even though the actual logic is handled by BOOSE's built-in If/Else classes, the factory needs to be able to create an ElseCommand object.

Syntax
C#
    internal class ElseCommand : CanvasCommand
Remarks
The 'else' command marks the alternative branch in an if-else statement. Processing is delegated to BOOSE's control flow system.
