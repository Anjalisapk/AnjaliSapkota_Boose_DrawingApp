Class RealCommand
Command for declaring a real (double) variable in BOOSE. Actual processing is handled by SimpleParser.
Syntax
C#
    internal class RealCommand : CanvasCommand
Remarks
This class exists primarily to satisfy the Factory Pattern, allowing the factory to recognize the "real" command. The Execute method is typically not called because SimpleParser intercepts real variable declarations.

Syntax:real variableName [= value]

Example:

C#
real pi = 3.14159
real temperature
