Class PenColourCommand
Implements the BOOSE pen / pencolour command. Changes the current drawing pen colour on the canvas.
Syntax
C#
    internal class PenColourCommand : CanvasCommand
Remarks
Supported formats:
  • Named colours — e.g., pen red, pen blue
  • RGB values — e.g., pen 255,0,0
Hex colours (#RRGGBB) are automatically generated internally when RGB values are used.
