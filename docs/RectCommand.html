Class RectCommand
Command for drawing a rectangle on the BOOSE canvas. Inherits from CommandTwoParameters and requires two parameters: width and height.
Syntax
C#
    public class RectCommand : CommandTwoParameters
Remarks

This command draws a rectangle at the current pen location using the specified width and height.

Example usage in BOOSE:

C#
rect 100 50      // Draws a rectangle 100 units wide and 50 units tall
rectangle 200 100 // Alternative keyword "rectangle"

Throws a CanvasException if:

  • Parameters are double values (not integers).
  • Width or height is less than 1.
