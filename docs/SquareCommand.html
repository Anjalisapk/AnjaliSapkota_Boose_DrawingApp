Class SquareCommand
Represents the BOOSE square command. Draws an unfilled square at the current pen position with specified side length.

Syntax:square size

Design Pattern: This class follows the Command Pattern, encapsulating a drawing operation as an object. It inherits from CommandOneParameter which is part of the BOOSE framework's command hierarchy.

Example Usage:

C#
moveto 150,150
square 80
This will draw a square with side length 80 at position (150,150).

A square is essentially a rectangle with equal width and height. This command internally uses the Rect(int, int, bool) method with width = height = size.

Syntax
C#
    public class SquareCommand : CommandOneParameter
Remarks
Related Design Patterns:
  • Command Pattern: Each drawing command is encapsulated as an object
  • Factory Pattern: Created by MyCommandFactory
  • Singleton Pattern: Uses AppCanvas singleton instance
  • Adapter Pattern: Adapts the Rect method to provide square functionality

Exception Condition
CanvasException Thrown when:
  • Double parameters are provided (only integers supported)
  • Size is less than 1
Remarks

Validation Rules:

  1. Only integer values are accepted (no doubles)
  2. Size must be greater than 0

Drawing Process: The square is drawn using Rect(int, int, bool) with both width and height set to the size parameter. This demonstrates the Adapter Pattern, where we adapt the rectangle drawing method to provide square-specific functionality.

Implementation Note: A square is a special case of a rectangle where width equals height. This design choice promotes code reuse and maintains consistency with the existing rectangle drawing functionality.
